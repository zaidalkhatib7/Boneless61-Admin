# Boneless 61 Admin

Production back-office CMS for Boneless 61.

## Features

- Admin authentication against the Boneless 61 Laravel API
- Branch, menu category, menu item, option group, option, and offer management
- English and Arabic interface with RTL support
- Responsive desktop and mobile layout
- Restaurant-themed animated background and custom cursor

## Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run lint
npm run build
```
